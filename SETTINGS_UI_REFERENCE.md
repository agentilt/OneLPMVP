## Design System

### Colors

**Light Mode:**
- Background: `bg-slate-50` (#f8fafc)
- Card background: `bg-white` (#ffffff)
- Text: `text-foreground` (dark gray)
- Borders: `border-slate-200` (#e2e8f0)
- Hover: `hover:bg-slate-50`

**Dark Mode:**
- Background: `bg-slate-950` (#020617)
- Card background: `bg-slate-900` (#0f172a)
- Text: `text-foreground` (light gray)
- Borders: `border-slate-800` (#1e293b)
- Hover: `hover:bg-slate-800/50`

### Typography

```
Page Title:       text-2xl font-semibold
Card Title:       text-lg font-semibold
Section Label:    text-xs uppercase tracking-wider font-medium
Body Text:        text-sm font-medium
Helper Text:      text-xs text-foreground/60
```

### Spacing

```
Card padding:         p-6 (24px)
Section padding:      p-6 (24px)
Between cards:        space-y-6 (24px)
Between items:        py-3 (12px vertical)
Border radius:        rounded-lg (8px)
```

### Interactive Elements

**Buttons:**
```css
Primary:   bg-accent text-white hover:bg-accent/90
Secondary: bg-blue-600 hover:bg-blue-700
Danger:    bg-red-600 hover:bg-red-700
```

**Badges:**
```css
Info:      bg-blue-100 text-blue-800 (dark: bg-blue-900/30 text-blue-300)
Success:   bg-green-100 text-green-800
Warning:   bg-yellow-100 text-yellow-800
Error:     bg-red-100 text-red-800
```

**Hover States:**
- Cards: `hover:bg-slate-50 dark:hover:bg-slate-800/50`
- Buttons: `hover:bg-accent/90`
- Rows: `hover:bg-slate-50 dark:hover:bg-slate-800/50`

## Responsive Behavior

### Desktop (1024px+)
- Two-column layout with sidebar navigation
- Full-width content cards
- All features visible

### Tablet (768px - 1023px)
- Sidebar navigation becomes narrower
- Content adjusts to available space
- Touch-friendly targets

### Mobile (<768px)
- Sidebar collapses to hamburger menu
- Single column layout
- Stack all content vertically
- Larger touch targets

## Accessibility

### Keyboard Navigation
- Tab through all interactive elements
- Arrow keys in toggle switches
- Enter to activate buttons
- Escape to close modals

### Screen Readers
- Proper ARIA labels on all buttons
- Form labels associated with inputs
- Status announcements for async actions
- Semantic HTML structure

### Color Contrast
- All text meets WCAG AA standards
- 4.5:1 minimum contrast for body text
- 3:1 minimum for large text
- Works in dark mode

## Animation & Transitions

```css
Tab switching:     instant (no animation for content speed)
Hover states:      transition-colors (150ms)
Toggle switches:   after:transition-all (300ms)
Success messages:  slide in from top (200ms)
```

## Enterprise Features

### Professional Data Display
- Table-like layouts for structured data
- Consistent row heights
- Hover states for interactivity
- Color-coded status indicators

### Status Indicators
- Green for active/success
- Blue for informational
- Yellow for warnings
- Red for errors/critical

### Action Patterns
- Primary action: Right-aligned, accent color
- Destructive action: Red color
- Inline actions: Text buttons in rows
- Form actions: Full-width at bottom

### Information Hierarchy
1. Page title (largest)
2. Section titles (large)
3. Field labels (small, uppercase)
4. Values (medium, prominent)
5. Helper text (smallest, muted)
